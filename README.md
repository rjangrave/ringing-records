# Ringing Records

Personal bell ringing records app for Richard J Angrave.

## Setup

1. Copy `config.js.example` to `config.js`
2. Fill in your Supabase credentials in `config.js`
3. Open `ringing-records-live.html` in your browser

## Files

- `ringing-records-live.html` — the main app (safe to commit to GitHub)
- `config.js` — your credentials (never committed, listed in .gitignore)
- `.gitignore` — tells GitHub to ignore config.js

## Updating the app

When Claude provides an updated HTML file:
1. Replace `ringing-records-live.html` on GitHub
2. Your `config.js` stays unchanged on your computer
3. No credential swapping needed!
